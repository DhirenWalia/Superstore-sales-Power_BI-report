# Superstore Sales Report — Power BI

A Power BI report built on the classic Superstore dataset.
This was a practice project to get comfortable with Power BI visuals,
slicers, and building a clean single-page report.

---

## What's in the Dashboard

| Visual | Shows |
|--------|-------|
| 3 Donut Charts | Profit by Category, Customer Segment, and Region |
| Pie Chart | Sales by Region |
| Line Chart | Profit trend over time (Year + Quarter) |
| Pivot Table | Sales by Product Sub-Category across Regions |
| 3 KPI Cards | Total Sales, Total Quantity, Total Profit |
| Slicer | Filter everything by State or Province |

## Tools Used

- Power BI Desktop

## Dataset

Classic Superstore dataset — US retail orders with sales, 
profit, quantity, category, region, and customer segment data.

## Key Insights

- Technology is the most profitable product category
- The West region leads in sales
- Profit dips noticeably in certain quarters — visible in the trend line
- Some sub-categories are consistently low in profit across all regions

## What I Learned

- Building a full report in Power BI from scratch
- Using slicers to make dashboards interactive
- Choosing the right chart type for each insight
- Formatting a clean, readable single-page layout

## How to Use

Download the `.pbix` file and open in Power BI Desktop (free from Microsoft).
Use the State slicer to filter all visuals at once.
